# Live Gift Calculator

Static website untuk kalkulator coin, level, dan referensi live TikTok.

## Deployment (Vercel + GitHub)

1. Push project ini ke repository GitHub.
2. Login ke Vercel dan klik `Add New Project`.
3. Import repository GitHub kamu.
4. Gunakan pengaturan default:
  - Framework Preset: `Other`
  - Build Command: kosong
  - Output Directory: kosong (root project)
5. Klik `Deploy`.

## Setelah Deploy

1. Pastikan semua halaman bisa diakses:
  - `/`
  - `/about.html`
  - `/contact.html`
  - `/privacy.html`
  - `/terms.html`
  - `/faq.html`
2. Cek file SEO:
  - `/robots.txt`
  - `/sitemap.xml`
  - `/ads.txt`
3. Jika domain production bukan `livegiftcalculator.com`, update URL pada `sitemap.xml` dan `robots.txt`.
